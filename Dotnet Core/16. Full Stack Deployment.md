# 🚀 1. Dockerize .NET App

## 📂 Root → `Dockerfile`

```dockerfile
# Build stage
FROM mcr.microsoft.com/dotnet/sdk:8.0 AS build
WORKDIR /app

COPY . .
RUN dotnet restore
RUN dotnet publish -c Release -o out

# Runtime stage
FROM mcr.microsoft.com/dotnet/aspnet:8.0
WORKDIR /app

COPY --from=build /app/out .

ENTRYPOINT ["dotnet", "MyApp.API.dll"]
```

---

## 📂 Root → `.dockerignore`

```text
bin/
obj/
node_modules/
.git/
```

---

## ▶️ Run

```bash
docker build -t myapp .
docker run -p 5000:80 myapp
```

## 📂 `docker-compose.yml`

```yaml
version: '3.8'

services:
  api:
    build: .
    ports:
      - "5000:80"
    depends_on:
      - redis
      - sqlserver

    image: redis:latest
    ports:
      - "6379:6379"

  sqlserver:
    image: mcr.microsoft.com/mssql/server
    environment:
      SA_PASSWORD: "Your_password123"
      ACCEPT_EULA: "Y"
    ports:
      - "1433:1433"
```

```csharp
{
```

```csharp
    public async Task GetUsers_ReturnsList()
    {
        var mockRepo = new Mock<IUserRepository>();

        mockRepo.Setup(x => x.GetAllAsync())
                .ReturnsAsync(new List<User>());

        var service = new UserService(mockRepo.Object);

        var result = await service.GetUsers();

        Assert.NotNull(result);
```

## 📂 Login API Call

```javascript
import axios from "axios";

const login = async () => {
  const res = await axios.post("/api/auth/login", {
    username: "admin",
    password: "123"
  });

  localStorage.setItem("token", res.data.token);
};
```

---

## 📂 Axios Interceptor

```javascript
axios.interceptors.request.use(config => {
  const token = localStorage.getItem("token");

  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }

  return config;
});
```

---

## 📂 Protected Route


# 🧠 1. What is Jenkins vs Jenkinsfile?

## 🔹 Jenkins (Tool)

👉 Software that runs CI/CD pipelines

* Like a **server**
* Runs builds, tests, deploys

---

## 🔹 Jenkinsfile

👉 A file inside your project

```text
Defines WHAT Jenkins should do
```

---

# 🔥 Real Analogy

```text
Jenkins → Chef 👨‍🍳  
Jenkinsfile → Recipe 📜
```

👉 Without chef → recipe useless
👉 Without recipe → chef doesn’t know what to cook
