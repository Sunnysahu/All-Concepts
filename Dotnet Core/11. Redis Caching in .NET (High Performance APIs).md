# 🚀 1. Redis Caching in .NET (High Performance APIs)

---

# 🧠 What is Caching?

👉 Store frequently used data in **memory** instead of DB

```text
Without Cache → API → DB (slow)
With Cache    → API → Cache (fast ⚡)
```

---

# 💣 Problem Without Cache

```csharp
public async Task<List<User>> GetUsers()
{
    return await _context.Users.ToListAsync(); // DB hit every time ❌
}
```

### Problems:

* Slow response ❌
* High DB load ❌
* Not scalable ❌

---

# ✅ With Cache

```text
1st Request → DB → Save in Cache  
Next Request → Cache (no DB) ⚡
```

---

# ⚙️ Step 1: Install Redis Package

```bash
dotnet add package Microsoft.Extensions.Caching.StackExchangeRedis
```

---

# ⚙️ Step 2: Configure Redis (Program.cs)

```csharp
builder.Services.AddStackExchangeRedisCache(options =>
{
    options.Configuration = "localhost:6379";
    options.InstanceName = "MyApp_";
});
```

---

# 🧪 Step 3: Use Cache

```csharp
using Microsoft.Extensions.Caching.Distributed;
using System.Text.Json;

public class UserService
{
    private readonly IDistributedCache _cache;
    private readonly AppDbContext _context;

    public UserService(IDistributedCache cache, AppDbContext context)
    {
        _cache = cache;
        _context = context;
    }

    public async Task<List<User>> GetUsers()
    {
        var cacheKey = "users";

        // 1. Try cache
        var cachedData = await _cache.GetStringAsync(cacheKey);

        if (cachedData != null)
        {
            return JsonSerializer.Deserialize<List<User>>(cachedData);
        }

        // 2. Fetch from DB
        var users = await _context.Users.ToListAsync();

        // 3. Save to cache
        var options = new DistributedCacheEntryOptions
        {
            AbsoluteExpirationRelativeToNow = TimeSpan.FromMinutes(5)
        };

        await _cache.SetStringAsync(cacheKey, JsonSerializer.Serialize(users), options);

        return users;
    }
}
```

---

# 🔥 Cache Expiration Types

```text
Absolute Expiration → expires after fixed time  
Sliding Expiration  → resets on each access
```

---

# ⚠️ Cache Invalidation (VERY IMPORTANT)

👉 When data changes → update cache

```csharp
await _cache.RemoveAsync("users");
```

# 💡 Pro Tips

* Don’t cache everything ❌
* Cache only **read-heavy data** ✅
* Always handle cache miss
