# 🚀 Next: MongoDB with .NET

# 🧠 What is MongoDB?

👉 NoSQL database (JSON-based)

```json
{
  "name": "Sunny",
  "age": 25
}
```

👉 Flexible schema (no strict tables)

---

# ⚙️ Step 1: Install Package

```bash
dotnet add package MongoDB.Driver
```

---

# ⚙️ Step 2: Configure

```csharp
using MongoDB.Driver;

public class MongoDbContext
{
    private readonly IMongoDatabase _database;

    public MongoDbContext()
    {
        var client = new MongoClient("mongodb://localhost:27017");
        _database = client.GetDatabase("MyAppDb");
    }

    public IMongoCollection<User> Users =>
        _database.GetCollection<User>("Users");
}
```

---

# 🧪 CRUD Example

---

## ➕ Create

```csharp
await _context.Users.InsertOneAsync(user);
```

---

## 📖 Read

```csharp
await _context.Users.Find(_ => true).ToListAsync();
```

---

## ✏️ Update

```csharp
var filter = Builders<User>.Filter.Eq(x => x.Id, id);
var update = Builders<User>.Update.Set(x => x.Name, "New");

await _context.Users.UpdateOneAsync(filter, update);
```

---

## ❌ Delete

```csharp
await _context.Users.DeleteOneAsync(x => x.Id == id);
```
