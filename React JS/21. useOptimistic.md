# React `useOptimistic` Hook (React 19)

`useOptimistic` lets you **show UI changes instantly** while an asynchronous update (like a server request) is in progress — improving perceived responsiveness. :contentReference[oaicite:1]{index=1}

---

## Syntax

```js
const [optimisticState, setOptimistic] = useOptimistic(actualState, reducer?);
````

* `optimisticState`: What your UI renders — the temporary state while an update is pending.
* `setOptimistic`: Call inside a transition to update state optimistically.
* Optional `reducer`: For calculating optimistic state from current + new value. ([React][1])

---

## Example: Optimistic “Like” Button

Instantly show a liked state while the server update happens:

```javascript
import React, { useState, useOptimistic, startTransition } from "react";

function LikeButton({ initialLiked }) {
  // Base state
  const [liked, setLiked] = useState(initialLiked);

  // Optimistic UI
  const [optimisticLiked, setOptimisticLiked] =
    useOptimistic(liked);

  function handleClick() {
    startTransition(async () => {
      // Immediately update UI
      setOptimisticLiked(!optimisticLiked);

      // Simulate server API call
      const result = await fakeServerToggle(!optimisticLiked);
      
      // Update real state after response
      setLiked(result);
    });
  }

  return (
    <button onClick={handleClick}>
      {optimisticLiked ? "❤️ Unlike" : "🤍 Like"}
    </button>
  );
}

async function fakeServerToggle(value) {
  await new Promise((r) => setTimeout(r, 1000));
  return value;
}

export default LikeButton;
```

---

## What Happens

1. User clicks the button → UI updates *optimistically* instantly.
2. Server request runs in the background.
3. When the request finishes, the **real state** (`liked`) updates and replaces the optimistic UI automatically. ([TechBoo][2])

---

## Notes

* You can derive `isPending` by comparing `optimisticState` versus `actualState`:

  ```js
  const isPending = optimisticState !== actualState;
  ```
* Optionally, supply a **reducer** to merge complex updates (like lists or counts). ([React][1])
