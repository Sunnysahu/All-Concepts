# React `useRef` Example: Auto-Focus Next Input

Use `useRef` to **control multiple input fields** and improve form usability.

---

## Example: Auto-focus Next Input

```javascript
import React, { useRef } from "react";

function FormInputs() {
  const firstNameRef = useRef(null);
  const lastNameRef = useRef(null);
  const emailRef = useRef(null);

  const handleKeyPress = (e, nextRef) => {
    if (e.key === "Enter" && nextRef.current) {
      nextRef.current.focus();
    }
  };

  return (
    <div>
      <input
        ref={firstNameRef}
        placeholder="First Name"
        onKeyDown={(e) => handleKeyPress(e, lastNameRef)}
      />
      <input
        ref={lastNameRef}
        placeholder="Last Name"
        onKeyDown={(e) => handleKeyPress(e, emailRef)}
      />
      <input
        ref={emailRef}
        placeholder="Email"
        onKeyDown={(e) => handleKeyPress(e, null)} // Last input
      />
    </div>
  );
}

export default FormInputs;
````

---

### ✅ Features:

1. **Three inputs**: First Name, Last Name, Email.
2. **Enter key moves focus** to the next field automatically.
3. Uses `useRef` for **direct DOM access** without re-renders.

---

### ⚡ Notes:

* `ref.current` stores the DOM element.
* Updating a ref does **not trigger re-rendering**.
* Perfect for **forms, focus management, or multi-step inputs**.

```
