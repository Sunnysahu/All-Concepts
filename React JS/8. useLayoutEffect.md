## useLayoutEffect

### 🧠 Theory
- Runs **synchronously after DOM mutations but before paint**  
- Blocks the browser from painting until the effect finishes  
- Useful for reading layout and synchronously making DOM updates  

### ✅ Use Case
- Measuring DOM elements (`getBoundingClientRect`)  
- Adjusting scroll positions  
- Synchronously applying styles or layout changes  

---

### 💻 Example: Basic useLayoutEffect
```javascript
import { useLayoutEffect, useRef } from "react";

export default function App() {
  const boxRef = useRef(null);

  useLayoutEffect(() => {
    const box = boxRef.current;
    console.log("Box width:", box.getBoundingClientRect().width);
  }, []);

  return <div ref={boxRef} style={{ width: 200, height: 100, background: "lightblue" }} />;
}
````

---

### 🔄 Behavior

* Runs **before the browser paints** → avoids flickering if you adjust DOM or styles
* Similar to `useEffect`, but **synchronous** and blocks rendering
