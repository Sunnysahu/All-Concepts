# React Select

### 🧠 Theory
- `react-select` is a library for **advanced dropdowns and multi-selects**  
- Provides **searchable, async, creatable, and styled select inputs**  
- Works with **single or multiple selections**  

### ✅ Use Case
- Dropdowns with many options  
- Multi-select or tag inputs  
- Searchable or async options  
- Custom styling and theming  

---

### 💻 Example: Basic Usage
```javascript
import React from "react";
import Select from "react-select";

const options = [
  { value: "chocolate", label: "Chocolate" },
  { value: "strawberry", label: "Strawberry" },
  { value: "vanilla", label: "Vanilla" }
];

export default function App() {
  const handleChange = (selectedOption) => {
    console.log("Selected:", selectedOption);
  };

  return <Select options={options} onChange={handleChange} placeholder="Select a flavor" />;
}
````

---

### 🔄 Features

* **Single select** and **multi-select** (`isMulti` prop)
* **Searchable** by default
* **Custom components**: styling, dropdown indicators, option rendering
* **Async loading** via `AsyncSelect`
* **Creatable options** via `CreatableSelect`

---

### ⚡ Tips

* Use `value` and `onChange` to **control selection state**
* For multi-select: `onChange` returns an array of selected options
* Works well with **React Hook Form** using a **custom controller**


``` javascript
import { Controller, useForm } from "react-hook-form";
import Select from "react-select";

<Controller
  name="flavor"
  control={control}
  render={({ field }) => (
    <Select {...field} options={options} />
  )}
/>
```
