# React `useTransition` Hook (React 18+)

The `useTransition` hook allows you to **mark state updates as “transitions”** for smoother UI updates, especially for heavy rendering tasks.  

---

## Syntax

```javascript
const [isPending, startTransition] = useTransition();
````

* `isPending`: `true` while the transition is ongoing.
* `startTransition`: function to wrap state updates that can be deferred.

---

## Example: Smooth List Filtering

```javascript
import React, { useState, useTransition } from "react";

function FilterList() {
  const [query, setQuery] = useState("");
  const [filteredList, setFilteredList] = useState([]);
  const [isPending, startTransition] = useTransition();

  const list = Array.from({ length: 10000 }, (_, i) => `Item ${i + 1}`);

  const handleChange = (e) => {
    const value = e.target.value;
    setQuery(value);

    // Heavy update wrapped in a transition
    startTransition(() => {
      const filtered = list.filter((item) =>
        item.toLowerCase().includes(value.toLowerCase())
      );
      setFilteredList(filtered);
    });
  };

  return (
    <div>
      <input
        type="text"
        placeholder="Search..."
        value={query}
        onChange={handleChange}
      />
      {isPending && <p>Loading...</p>}
      <ul>
        {filteredList.map((item) => (
          <li key={item}>{item}</li>
        ))}
      </ul>
    </div>
  );
}

export default FilterList;
```

---

### ✅ Key Points

1. Wrap **expensive or non-urgent updates** with `startTransition` to keep the UI responsive.
2. `isPending` can show a **loading indicator** while the transition is in progress.
3. Ideal for **filtering large lists, rendering complex components, or animations**.

---

### ⚡ Notes

* Updates inside `startTransition` are **low-priority**, so urgent UI (like typing) remains smooth.
* Works only in **React 18+**.
