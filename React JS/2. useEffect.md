## useEffect

### 🧠 Theory
- Handles side effects  
- Runs after render  
- Dependency array controls execution  

### ✅ Use Case
- API calls, subscriptions  

### 💻 Example
```javascript
import { useEffect, useState } from "react";

function Users() {
  const [users, setUsers] = useState([]);

  useEffect(() => {
    fetch("/api/users")
      .then(res => res.json())
      .then(setUsers);
  }, []);

  return users.map(u => <div key={u.id}>{u.name}</div>);
}
