# Lazy Loading + Suspense+

### 🧠 Theory
- Enables **code splitting** in React  
- Components are **loaded only when needed**  
- `Suspense` provides a **fallback UI** while the lazy component loads  

### ✅ Use Case
- Performance optimization  
- Large components or routes  
- Reducing initial bundle size  

---

### 💻 Example: Basic Lazy Loading
```javascript
import React, { Suspense } from "react";

const LazyComp = React.lazy(() => import("./Comp"));

export default function App() {
  return (
    <div>
      <h1>My App</h1>
      <Suspense fallback={<p>Loading...</p>}>
        <LazyComp />
      </Suspense>
    </div>
  );
}
````

---

### 🔄 How It Works

1. `React.lazy()` dynamically imports the component
2. `Suspense` displays the fallback while the component loads
3. Once loaded, the lazy component **renders normally**

---

### ⚡ Tip

* Use for **routes, heavy UI components, charts, or dashboards**
* Combine with **React Router** for route-level lazy loading
