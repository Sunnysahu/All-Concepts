# React Hook Form – Full Feature Example with Props

### 🧠 Theory
- `react-hook-form` provides multiple **helper methods and props** for form handling  
- Key features: **validation, error handling, value watching, resetting, default values, etc.**  

---

### 💻 Example: Using Various Props and Methods
```javascript
import React from "react";
import { useForm } from "react-hook-form";

export default function App() {
  const {
    register,
    handleSubmit,
    watch,
    setValue,
    reset,
    getValues,
    formState: { errors, isDirty, isSubmitting }
  } = useForm({
    defaultValues: {
      firstName: "",
      lastName: "",
      email: "",
      password: "",
      subscribe: false,
      gender: "",
      country: "",
      bio: ""
    }
  });

  const onSubmit = (data) => {
    console.log("Form Data:", data);
  };

  // Watch example
  const watchAll = watch(); // watch all fields
  const watchEmail = watch("email"); // watch specific field

  return (
    <form onSubmit={handleSubmit(onSubmit)} style={{ display: "grid", gap: "10px", maxWidth: "400px" }}>
      <input {...register("firstName", { required: "First Name required" })} placeholder="First Name" />
      {errors.firstName && <span>{errors.firstName.message}</span>}

      <input {...register("lastName", { required: "Last Name required" })} placeholder="Last Name" />
      {errors.lastName && <span>{errors.lastName.message}</span>}

      <input
        type="email"
        {...register("email", { required: "Email required", pattern: /^\S+@\S+$/i })}
        placeholder="Email"
      />
      {errors.email && <span>Email is invalid</span>}

      <input type="password" {...register("password", { minLength: 6 })} placeholder="Password" />
      {errors.password && <span>Password must be at least 6 characters</span>}

      <label>
        <input type="checkbox" {...register("subscribe")} />
        Subscribe
      </label>

      <label>
        <input type="radio" value="male" {...register("gender", { required: true })} /> Male
      </label>
      <label>
        <input type="radio" value="female" {...register("gender", { required: true })} /> Female
      </label>
      {errors.gender && <span>Gender is required</span>}

      <select {...register("country", { required: true })}>
        <option value="">Select Country</option>
        <option value="usa">USA</option>
        <option value="india">India</option>
        <option value="uk">UK</option>
      </select>
      {errors.country && <span>Country is required</span>}

      <textarea {...register("bio")} placeholder="Short Bio" />

      <div>
        <button type="submit" disabled={isSubmitting}>Submit</button>
        <button type="button" onClick={() => reset()}>Reset</button>
        <button type="button" onClick={() => setValue("firstName", "John")}>Set First Name</button>
      </div>

      <div>
        <strong>Watched Email:</strong> {watchEmail}
      </div>
      <div>
        <strong>Form Values:</strong> {JSON.stringify(getValues())}
      </div>
      <div>
        <strong>Form Dirty:</strong> {isDirty ? "Yes" : "No"}
      </div>
    </form>
  );
}
````

---

### 🔄 Key Properties & Methods

* `register` – register input fields
* `handleSubmit` – handle form submission
* `watch` – watch specific/all fields
* `setValue` – programmatically set field value
* `getValues` – get current form values
* `reset` – reset the form
* `formState` – contains `errors`, `isDirty`, `isSubmitting`, `isValid`

---

### ⚡ Tips

* Use `defaultValues` to prefill forms
* Combine with **Yup** or **Zod** for schema validation
* Use `isDirty` to **enable/disable submit button**
